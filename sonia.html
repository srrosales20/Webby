What is Typography?

Typography is a design discipline that uses typeface to organize, create readable, usable, and user-friendly interfaces and experiences. By arranging typefaces with different fonts, sizes, and spacing we are attracting users and creating a positive experience.

What is Color Psychology?

Color psychology is the study of how different colors impact human behavior. Every color has a different meanings, psycological effect, connotations. Color psycology is an important aspect of UI because it helps set the tone for websites/projects..
